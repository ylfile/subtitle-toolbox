# YLFile字幕工具箱 v2.1

有料视界（YLFile）字幕组一体化工作流工具：**MKV 规范命名 → 提取字幕 → 黑边测量 → 生成 ASS → 音轨处理 → 压制**。

- 微博：[YLFile](https://weibo.com)
- 官网：https://ylfile.com

## 功能一览

| 步骤 | 说明 |
|------|------|
| **规范命名** | 将 MKV/MP4 统一重命名为 SxxExx.mkv / SxxExx.mp4 |
| **提取字幕** | 从 MKV 提取原版/中文字幕轨，繁体自动转简体；**支持手动指定原版语言** |
| **黑边测量** | 手动点击 + 批量自动检测，每部剧只需测一次，结果自动保存 |
| **生成 ASS** | 双语/仅中文字幕，自动定位黑边区域；**字幕组信息可编辑**；**生成后可选自动转换为 Aegisub 格式**以修复压制重叠 |
| **音频处理** | 按文件夹语言标识或手动指定原版语言，保留对应音轨，去掉多余音轨 |

## 快速开始

1. 下载 `YLFile字幕工具箱_v2.1.0.exe`
2. 双击运行
3. 顶部目录栏选择字幕根目录
4. 按工作流操作：**规范命名 → 提取字幕 → 测黑边 → 生成 ASS**

> 详细说明见 `使用说明.txt`。

## 工作流

```
规范命名（统一文件名）
    ↓
提取字幕（从 MKV 提取 SRT，可选手动指定原版语言）
    ↓
黑边测量（每部剧测一次，结果自动保存到 config.json）
    ↓
生成 ASS（双语或仅中文，自动定位黑边区域，字幕组信息可配置）
    ↓
[可选] 音频处理（去多余音轨）
```

所有功能共享顶部**全局根目录**，切换标签页无需重复选择目录。

## 字幕目录约定

```
字幕根目录/
  剧名A/
    S01E01.mkv
    S01E01原版.srt          # 提取后
    S01E01中文版.srt        # 提取后
    S01E01双语-1080p.ass   # 生成后
  剧名B/
    ...
```

## 源码运行

```bash
pip install -r requirements.txt
python main.py
```

需将 `mkvmerge.exe`、`mkvextract.exe` 放在程序根目录；`opencc/` 与 `share/opencc/` 已随仓库提供。

## 依赖

- Python 3.12+ / tkinter
- OpenCV（自动黑边检测）
- OpenCC（繁转简）
- pysubs2（ASS 格式转换）
- mkvmerge / mkvextract（MKV 处理）

## 打包 exe

```bash
pip install pyinstaller
pyinstaller "YLFile字幕工具箱.spec"
```
