将以下文件放在程序根目录（与 main.py 或打包后的 exe 同级）：

1. ffmpeg.exe、ffprobe.exe
   推荐 Gyan 完整版：https://www.gyan.dev/ffmpeg/builds/
   （ffprobe 与 ffmpeg 放在同一目录）

2. mkvmerge.exe、mkvextract.exe
   来自 MKVToolNix：https://mkvtoolnix.download/

3. opencc 文件夹（本仓库已含 opencc\ 与 share\opencc\）

打包 exe：安装 Python 依赖后运行 build.bat
